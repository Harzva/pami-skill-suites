# Reviewer-risk map

| Risk | Trigger | Mitigation |
|---|---|---|
| Misrouted venue | User says PR but means Pattern Recognition, or PAMI vs Pattern Recognition is confused | Confirm publisher family and route accurately. |
| Unsupported official rule | Skill states a rule without source | Convert to verification item tied to official pages. |
| Template misuse | Example paper text is reused | Replace with section skeleton and rewrite from user evidence. |
| Overclaiming | Abstract claims more than experiments show | Soften, add missing experiments, or mark as hypothesis. |
| Missing package materials | Highlights, Index Terms, declarations, or source files missing | Add submission-package checklist. |
