# IEEE Journal Taxonomy

## Core adapters in this repository

- `pami-writing`: pattern analysis, computer vision, machine learning.
- `tip-writing`: image processing and computational imaging.
- `tmi-writing`: medical imaging.
- `tnnls-writing`: neural networks and learning systems.
- `tmm-writing`: multimedia.
- `tkde-writing`: data engineering and knowledge discovery.
- `tifs-writing`: information forensics and security.
- `jbhi-writing`: biomedical and health informatics.
- `tgrs-writing`: geoscience and remote sensing.
- `tsp-writing`: signal processing.

## Routing rule

If the task is broad IEEE style, start with `ieee-writing`. If a target journal is named, use the closest adapter after running the core IEEE checks.
