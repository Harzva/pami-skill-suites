# IEEE Manuscript Readiness Rubric

Score each item 0–2.

| Criterion | 0 | 1 | 2 |
|---|---|---|---|
| Problem definition | vague | partially clear | precise and motivated |
| Novelty | unclear | incremental but stated | clear technical contribution |
| Method detail | insufficient | mostly reproducible | reproducible and well motivated |
| Experiments | weak | adequate | strong, fair, comprehensive |
| Ablations | missing | limited | addresses design choices |
| Related work | shallow | covers main work | positions gap precisely |
| Reproducibility | poor | partial | strong details/code/data plan |
| Writing | promotional/vague | readable | concise and technical |

Interpretation:

- 0–7: high rejection risk.
- 8–12: revise before submission.
- 13–16: plausible submission-ready draft.
