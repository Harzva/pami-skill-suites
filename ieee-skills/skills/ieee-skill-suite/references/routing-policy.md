# Routing policy

The router applies a three-layer decision:

1. **Journal layer**: target journal or publisher family.
2. **Component layer**: manuscript part such as related work, table, figure, caption, method, experiment, declarations, or references.
3. **Submission layer**: cover letter, revision response, author agreement, copyright/license, open access, final files, proof correction, or official-source checks.

The router should select the smallest useful skill chain. It should not load every skill in the suite.
