# Main-Figure Website + Multimodal RAG Roadmap

Target: v2.0.0.

Goal: build a standalone website and multimodal RAG knowledge base with at least 100 legally sourced OA main figures for IEEE / journal-skill-suite research-figure design analysis.

## Scope

- Display main figures as a curated gallery where legally permitted.
- Store safe structural descriptions and metadata for RAG.
- Link visual patterns to motivation, method, experiment, limitations, and caption routines.

## Safety

Do not ingest Sci-Hub, pirated, paywalled, or copyright-unclear PDFs. Do not expose extracted visuals for reuse without checking license and attribution requirements.
