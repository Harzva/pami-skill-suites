# Official Source Rubric

Applies to: IEEE journal skill-suite evaluation tasks.

## Scoring scale

Each criterion is scored from 0 to 5.

- 0 = absent or unsafe
- 1 = present but mostly unusable
- 2 = partial and weak
- 3 = acceptable
- 4 = strong
- 5 = release-quality

## Criteria
1. **publisher official rule separation.** The output must be specific, checkable, and aligned with the target journal task.
2. **journal official rule separation.** The output must be specific, checkable, and aligned with the target journal task.
3. **repository recommendation labeling.** The output must be specific, checkable, and aligned with the target journal task.
4. **template-paper observation labeling.** The output must be specific, checkable, and aligned with the target journal task.
5. **mutable rule live-check flagging.** The output must be specific, checkable, and aligned with the target journal task.

## Automatic gate

A sample output passes the static gate when it includes selected skills/modules, required warnings, a reviewer-risk checklist, and the structured output contract required by the task.

## Human review note

This rubric is a benchmark scaffold. It does not replace expert editorial review or live publisher/journal verification.
