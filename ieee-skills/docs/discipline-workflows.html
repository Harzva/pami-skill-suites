Workflow pattern

  1. Choose an optional discipline preset.
  2. Route through the compact suite router.
  3. Select only necessary journal/component/submission fragments.
  4. Apply official-source and copyright safety warnings.
  5. Return structured output with reviewer-risk checks.

Example

Use the ieee-skill-suite with the computer-vision preset to upgrade related work, ablation table, and figure captions. Do not expose advanced skills. Return selected modules and reviewer-risk checklist.

Non-goals

Presets are not new default skills. They are not a replacement for live journal instructions, publishing agreements, copyright/license policies, or official submission systems.

Unofficial project. Not affiliated with IEEE. Verify live publisher and journal pages before real submission.
