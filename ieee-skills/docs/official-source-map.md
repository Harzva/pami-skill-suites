## Maintenance rule

Before adding a new rule to any skill, classify it as:

- official rule;
- official template/style guidance;
- repository convention;
- open-access template-paper pattern;
- unverified target-journal item.

Rules that may change must stay as verification checklist items instead of hard-coded promises.
