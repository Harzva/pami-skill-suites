# Manuscript Component Coverage

This repository now covers manuscript details at component level.

| Area | Covered by |
|---|---|
| Title / authors / metadata | `ieee-title-author-metadata` |
| Abstract | `ieee-abstract` |
| Keywords / Index Terms | `ieee-index-terms` |
| Introduction | `ieee-introduction` |
| Related work | `ieee-related-work` |
| Method / algorithm / theory | `ieee-method-details`, `ieee-algorithm-pseudocode`, `ieee-theory-proof` |
| Experiments / baselines / metrics | `ieee-experimental-design`, `ieee-baseline-selection`, `ieee-metrics-evaluation` |
| Ablations / statistics | `ieee-ablation-study`, `ieee-statistical-reporting` |
| Tables | `ieee-table` |
| Figures | `ieee-figure` |
| Captions | `ieee-caption` |
| Discussion / limitations / conclusion | `ieee-discussion`, `ieee-limitations`, `ieee-conclusion` |
| Supplementary / data / code | `ieee-supplementary-files`, `ieee-data-code-availability` |
| Author agreement / copyright / OA | `ieee-author-agreement`, `ieee-copyright-license`, `ieee-open-access-decision` |
| Response / revision | `ieee-reviewer-response-matrix`, `ieee-revision-diff` |
