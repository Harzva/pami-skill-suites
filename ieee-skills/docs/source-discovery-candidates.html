OA Source Discovery Candidates

Release: v2.3.0

The candidate registry upgrades pending 100-paper slots into auditable source-discovery records with proposed DOI, source URL, license candidate, OA status, risk, and extraction readiness.

Safety policy

This workflow is offline-safe by default. It does not download papers, bypass paywalls, use Sci-Hub, or treat pending candidates as approved OA sources.

Required artifacts

data/source_discovery_candidates.json
data/source_discovery_candidates.yaml

Maintainer command

python scripts/discover_oa_candidate_sources.py
