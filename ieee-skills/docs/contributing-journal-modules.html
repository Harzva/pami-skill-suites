Contribution workflow

Use templates/new-journal-fragment.md when adding a journal module. Keep detailed journal rules inside skills/ieee-skill-suite/static/journals/; do not add new default-visible skills unless the module is a broadly reusable macro skill.

Required source fields

  • Journal homepage
  • Author instructions / Guide for Authors
  • Submission system
  • Manuscript template
  • Publishing agreement
  • Copyright / license
  • Open access
  • Ethics / integrity
  • AI disclosure policy
  • Data / code availability
  • Permissions

Required metadata

  • last_live_checked
  • source_status
  • needs_reverification
  • official_link_health
  • Live verification warning

Release gate

python scripts/validate_sources.py
python scripts/check_official_links_live.py --live
python scripts/report_stale_sources.py

Unofficial project. Verify live publisher and journal pages before real submission.
