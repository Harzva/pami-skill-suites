# Comparison with nature-skills

This project aims to match the product maturity of a polished academic skill repository while targeting IEEE journals rather than Nature/CNS-style workflows.

## Shared product principles

- installable skill units
- supporting references and resources
- product documentation
- examples and workflow guidance

## Distinct design choices

- compact-by-default macro skills
- journal / component / submission fragments
- official-source provenance matrix
- lawful paper-template mining
- advanced skills opt-in
