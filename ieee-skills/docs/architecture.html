Architecture

skills/                  compact default macro skills
skills/ieee-skill-suite/static/  journals, components, submission fragments
skills-advanced/         optional independent skills
resources/               registries and official-source metadata
paper_templates/         lawful corpus and structure maps
evals/                   benchmark tasks and sample outputs
plugins/                 plugin metadata
dist/                    generated packages

This design keeps high detail inside fragments while exposing only a small number of default macro skills.

IEEE Skills is unofficial and not affiliated with IEEE. Verify current official instructions before real submission.
