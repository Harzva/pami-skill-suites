# Product Parity Notes for v0.5.0

This release improves product completeness by adding:

- suite-level router skill;
- journal, component, and submission routers;
- registry-driven route planning;
- compact one-skill distribution;
- full-suite distribution manifest;
- route validation scripts;
- GitHub Pages router and compact-suite pages;
- official-source registry for skill architecture and publishing checks.

The product goal is a repository that behaves like a real installable skill suite rather than a prompt dump.
