Promote Journal Candidates

Context-safe journal skill-suite documentation.

Promotion workflow

L0 candidates are not installed as default skills. Use the promotion script to create a staging draft, then verify official sources before importing it as a suite/static fragment or optional advanced skill.

python scripts/promote_journal_candidate.py --pack computer-vision --journal-id example-journal
python scripts/validate_pack_maturity.py

Promotion must preserve context safety: no candidate may be added to the default skills/ listing automatically.

Required evidence

  • Official homepage
  • Author instructions / Guide for Authors
  • Full official-source matrix
  • Source-refresh metadata
  • Live verification warning
