CLI preview

Use the preview script to see which suite, journals, and components a preset would choose. The script is static and safe; it does not do live official-source verification.

python scripts/preset_router_preview.py --list
python scripts/preset_router_preview.py --preset computer-vision --task "PAMI related work figure caption audit"

Output contract

  • selected preset
  • suite router
  • candidate journals
  • selected component modules
  • official-source warning
  • context policy

Unofficial project. Not affiliated with IEEE. Verify live publisher and journal pages before real submission.
