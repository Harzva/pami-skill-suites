How to use this page

Use the JSON/YAML assets as a transparent debugging layer for scientific-figure retrieval. Each trace links the user query to expected tags, seed metadata records, blocked RAG modes, failure-mode candidates, query rewrites, and human-evaluation placeholders.

Do not use as image reuse permission

These assets do not grant permission to copy, redraw, embed, publish, or publicly display source figures. They only support safe structural analysis and metadata-only retrieval.
