# Security Policy

## Supported version

v2.7.0 is the current public release.

## Reporting

Please report security-sensitive issues privately through repository maintainers when available. Do not include private manuscripts, unpublished research data, reviewer identities, confidential peer-review material, or credentials in public issues.

## Data safety

This repository should not store private manuscripts, confidential reviews, credentials, API keys, or non-public datasets.
