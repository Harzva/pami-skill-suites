# Corpus license and copyright notes

This corpus is built for structural study only.

## Allowed source categories

- Open-access papers with source/license metadata.
- CC BY / CC BY-NC / publisher-permitted OA PDFs.
- Official publisher/template/manual PDFs.
- Author or institutional repository OA versions.

## Prohibited sources

- Sci-Hub or similar unauthorized mirrors.
- Pirated PDFs.
- Paywalled PDFs without permission.
- Copyright-unknown PDFs.
- Full-text copying without authorization.

## Use restrictions

Do not copy text, figures, tables, captions, equations, layouts, paper-specific claims, benchmark interpretations, or medical/clinical claims from the bundled PDFs. Use only high-level structural observations and rewrite from the user's evidence.

## Live verification

Before public redistribution or real submission support, re-check each source URL and license. Corpus metadata records the source state used by this repository, but publisher pages and repository licenses can change.
