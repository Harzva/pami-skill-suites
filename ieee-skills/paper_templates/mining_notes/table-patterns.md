# Table Patterns

This note summarizes reusable structural lessons from the lawful `IEEE` template-paper corpus. It does not quote or copy source text.

## Corpus coverage

- `tpami-2023-transformers-survey` — IEEE TPAMI, 2023: Survey structure, taxonomy-driven related work, dense sectioning, visual overview tables, TPAMI-style synthesis.
- `tip-2022-see360` — IEEE TIP, 2022: Image-processing method exposition, visual-result comparison, ablation rhythm, quantitative/qualitative evidence pairing.
- `tmi-2020-lesion-localization` — IEEE TMI, 2020: Medical-imaging motivation, dataset/clinical-context framing, interpretability evidence, risk-aware experimental presentation.
- `tnnls-2023-sparce` — IEEE TNNLS, 2023: Learning-systems method narrative, theoretical/method motivation, equations plus experiments.
- `tnnls-2024-mediterranean` — IEEE TNNLS, 2024: Compact theory-to-experiment flow, mathematical contribution framing, complexity/efficiency storyline.

## General pattern

- Use tables for exact comparison and ablation evidence.
- Keep header hierarchy readable.
- Explain every metric, marker, and bold/underline convention.
