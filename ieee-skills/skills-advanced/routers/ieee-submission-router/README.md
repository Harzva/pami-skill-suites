# Ieee Submission Router

This is an installable router skill in the IEEE Skills repository. Copy the full folder, not only `SKILL.md`, because the router uses references, examples, checklists, and scripts.

## Purpose

route submission workflows such as cover letter, author agreement, copyright/license, open access, revision response, final files, proof correction, and official-source checks.

- `references/task-to-skill-map.md`

## Status

Experimental and unofficial. Always verify live publisher and journal instructions before submission.
