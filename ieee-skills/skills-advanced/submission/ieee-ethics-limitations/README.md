# IEEE Ethics + Limitations

**Skill name:** `ieee-ethics-limitations`  
**Family:** `ieee`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **limitations, responsible AI, dataset bias, human/medical/security risk, and dual-use disclosure prompts**.

## Typical commands

```text
Use ieee-ethics-limitations to audit this abstract for venue fit and reviewer risk.
Use ieee-ethics-limitations to rewrite my introduction in a more technical journal style.
Use ieee-ethics-limitations to produce a claim-evidence map and missing-experiment list.
Use ieee-ethics-limitations to draft a response letter from these reviewer comments.
```
