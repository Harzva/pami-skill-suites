# IEEE AI Disclosure + Integrity

**Skill name:** `ieee-ai-disclosure-integrity`  
**Family:** `ieee`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **transparent AI-assisted writing, plagiarism avoidance, citation integrity, and author-responsibility reminders**.

## Typical commands

```text
Use ieee-ai-disclosure-integrity to audit this abstract for venue fit and reviewer risk.
Use ieee-ai-disclosure-integrity to rewrite my introduction in a more technical journal style.
Use ieee-ai-disclosure-integrity to produce a claim-evidence map and missing-experiment list.
Use ieee-ai-disclosure-integrity to draft a response letter from these reviewer comments.
```
