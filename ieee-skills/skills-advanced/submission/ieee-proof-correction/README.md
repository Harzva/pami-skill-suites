# `ieee-proof-correction` skill

Component skill for **proof review, correction discipline, author queries, metadata checks, and post-acceptance risk control** in IEEE journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.
