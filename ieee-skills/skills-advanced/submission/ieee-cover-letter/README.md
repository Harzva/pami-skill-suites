# IEEE Cover Letter

**Skill name:** `ieee-cover-letter`  
**Family:** `ieee`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **journal submission cover letter**.

## Typical commands

```text
Use ieee-cover-letter to audit this abstract for venue fit and reviewer risk.
Use ieee-cover-letter to rewrite my introduction in a more technical journal style.
Use ieee-cover-letter to produce a claim-evidence map and missing-experiment list.
Use ieee-cover-letter to draft a response letter from these reviewer comments.
```

This skill is not affiliated with any publisher, society, editor, or journal. Always verify the latest official author instructions for the target journal before submission.
