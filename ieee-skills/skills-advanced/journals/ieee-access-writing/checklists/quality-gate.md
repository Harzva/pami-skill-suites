# Quality Gate

Score each item from 0 to 2.

| Item | 0 | 1 | 2 | Score |
|---|---|---|---|---|
| Venue fit | wrong or unclear | plausible | strong fit |  |
| Novelty | vague | partially clear | explicit and defensible |  |
| Evidence | unsupported | partly supported | claim-evidence aligned |  |
| Experiments | incomplete | acceptable | comprehensive and fair |  |
| Reproducibility | missing | partial | sufficient details |  |
| Limitations | absent | generic | specific and honest |  |
| Style | informal or hype | acceptable | precise and journal-ready |  |
| Submission package | missing | partial | complete/checkable |  |

## Interpretation

- 0–6: not submission-ready.
- 7–11: promising but risky.
- 12–16: strong draft; run official instruction checks before submission.
