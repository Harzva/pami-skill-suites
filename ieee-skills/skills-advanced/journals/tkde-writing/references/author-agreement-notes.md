# Author Agreement Notes: IEEE Transactions on Knowledge and Data Engineering

IEEE publishing agreement decisions are generally post-acceptance decisions. Before publication, IEEE asks authors to complete a publishing agreement. The repository tracks two main official verification paths:

- IEEE Copyright Form / traditional route.
- IEEE open access route with Creative Commons license options, where applicable.

## Checks for this skill

- Verify the current IEEE Author Center publishing agreement page.
- Verify whether the target publication is hybrid or fully open access.
- Verify current APC, institutional agreement, funder mandate, and license requirements.
- Verify whether reused figures, tables, text, datasets, or code require permission or attribution.
- Never state that a license has been selected, a form has been signed, or a fee has been paid unless the author provides that information.
