# Example Prompts for `tase-writing`

## Abstract rewrite

```text
Use tase-writing. Rewrite this abstract for the target journal. Preserve all results exactly. Produce: revised abstract, claim-evidence map, reviewer-risk notes, and missing evidence list.
```

## Introduction diagnosis

```text
Use tase-writing. Diagnose whether this introduction clearly states problem, gap, novelty, and contribution. Return a paragraph-by-paragraph revision plan and a rewritten version.
```

## Experiment audit

```text
Use tase-writing. Audit these experiment tables for baseline fairness, ablations, metrics, statistical support, and missing failure analysis.
```

## Reviewer response

```text
Use tase-writing. Draft a respectful response letter. For each reviewer comment, produce response strategy, manuscript change, exact response text, and remaining risk.
```

## Full package audit

```text
Use tase-writing. Audit this submission package. Check manuscript structure, figures/tables, citations, reproducibility, limitations, declarations, and official-instruction verification items.
```
