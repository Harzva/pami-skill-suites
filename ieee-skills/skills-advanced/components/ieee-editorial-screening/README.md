# IEEE Editorial Screening

**Skill name:** `ieee-editorial-screening`  
**Family:** `ieee`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **desk-rejection risk, scope fit, novelty, formatting, ethics, and completeness audit**.

## Typical commands

```text
Use ieee-editorial-screening to audit this abstract for venue fit and reviewer risk.
Use ieee-editorial-screening to rewrite my introduction in a more technical journal style.
Use ieee-editorial-screening to produce a claim-evidence map and missing-experiment list.
Use ieee-editorial-screening to draft a response letter from these reviewer comments.
```
