# `ieee-results-reporting` skill

Component skill for **results narrative, table/figure coordination, statistical strength, and non-cherry-picked reporting** in IEEE journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.

## Typical use

```text
@ieee-results-reporting Rewrite this section for IEEE journal style and return a claim-evidence audit.
```
