# `ieee-appendix` skill

Component skill for **appendices, extended derivations, extra tables, and material that should not interrupt the main paper** in IEEE journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.

## Typical use

```text
@ieee-appendix Rewrite this section for IEEE journal style and return a claim-evidence audit.
```
