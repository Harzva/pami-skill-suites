# Reviewer Risk Map: IEEE Introduction

| Risk | Symptom | Fix |
|---|---|---|
| Overclaim | Strong claim without evidence | Add evidence or soften wording |
| Underspecified method | Missing implementation or protocol details | Add reproducibility details |
| Weak positioning | Prior work not grouped or contrasted | Build a taxonomy and delta table |
| Visual confusion | Figure/table/caption mismatch | Rewrite caption and align text references |
| Policy uncertainty | Unverified author agreement or submission claim | Add an official-source verification item |
