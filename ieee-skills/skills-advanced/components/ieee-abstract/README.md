# `ieee-abstract` skill

Component skill for **abstract drafting, compression, claim discipline, and contribution evidence alignment** in IEEE journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.

## Typical use

```text
@ieee-abstract Rewrite this section for IEEE journal style and return a claim-evidence audit.
```
