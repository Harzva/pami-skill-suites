# `ieee-method-details` skill

Component skill for **method section completeness, implementation clarity, architectural motivation, and reproducibility details** in IEEE journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.

## Typical use

```text
@ieee-method-details Rewrite this section for IEEE journal style and return a claim-evidence audit.
```
