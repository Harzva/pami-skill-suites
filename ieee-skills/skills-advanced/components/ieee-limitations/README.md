# `ieee-limitations` skill

Component skill for **honest limitations, failure cases, boundary conditions, ethics, and future work without self-sabotage** in IEEE journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.

## Typical use

```text
@ieee-limitations Rewrite this section for IEEE journal style and return a claim-evidence audit.
```
