# IEEE LaTeX Template Audit

**Skill name:** `ieee-latex-template-audit`  
**Family:** `ieee`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **IEEE template-readiness, source package hygiene, equations, captions, and references**.

## Typical commands

```text
Use ieee-latex-template-audit to audit this abstract for venue fit and reviewer risk.
Use ieee-latex-template-audit to rewrite my introduction in a more technical journal style.
Use ieee-latex-template-audit to produce a claim-evidence map and missing-experiment list.
Use ieee-latex-template-audit to draft a response letter from these reviewer comments.
```
