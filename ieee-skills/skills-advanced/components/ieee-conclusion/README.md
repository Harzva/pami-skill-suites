# `ieee-conclusion` skill

Component skill for **conclusion concision, contribution recap, implications, and no-new-claim discipline** in IEEE journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.

## Typical use

```text

```
