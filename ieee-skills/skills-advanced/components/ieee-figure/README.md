# `ieee-figure` skill

Component skill for **figures, panels, visual evidence, resolution/readability, accessibility, and text-reference order** in IEEE journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.

## Typical use

```text
@ieee-figure Rewrite this section for IEEE journal style and return a claim-evidence audit.
```
