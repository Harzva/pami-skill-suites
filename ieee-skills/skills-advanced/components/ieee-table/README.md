# `ieee-table` skill

Component skill for **submission-ready tables, editable table text, caption logic, notes, abbreviations, and non-duplicative presentation** in IEEE journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.

## Typical use

```text
@ieee-table Rewrite this section for IEEE journal style and return a claim-evidence audit.
```
