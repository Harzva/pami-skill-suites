# Journal / workflow profile

This skill is not tied to one journal. It works as a meta-skill for `ieee-skills` and must route to the correct core skill or journal adapter.

## Source hierarchy

1. Current target-journal author instructions.
2. IEEE Author Center and target publication instructions.
3. Official template/style/declaration pages.
4. Repository shared policies.
5. Bundled CC-BY/open-access papers used only for structural study.

## Non-goals

- Do not replace official submission portals.
- Do not infer page charges or limits from memory.
- Do not use example papers as reusable prose.
