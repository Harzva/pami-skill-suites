# `ieee-discussion` skill

Component skill for **discussion section interpretation, implications, comparison to prior work, and evidence-bounded claims** in IEEE journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.

## Typical use

```text
@ieee-discussion Rewrite this section for IEEE journal style and return a claim-evidence audit.
```
