# `ieee-caption` skill

Component skill for **figure and table captions, legends, abbreviation expansion, symbol explanation, and standalone interpretability** in IEEE journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.

## Typical use

```text
@ieee-caption Rewrite this section for IEEE journal style and return a claim-evidence audit.
```
