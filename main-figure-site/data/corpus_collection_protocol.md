# Main-Figure Corpus Collection Protocol

## Goal

Build a 100+ paper main-figure corpus for scientific figure design analysis and multimodal RAG.

## Allowed sources

- Open-access papers.
- CC BY / CC BY-NC / explicitly licensed versions.
- Publisher-permitted OA PDFs.
- Author or institution repository OA versions.

## Prohibited sources

- Sci-Hub.
- Pirated PDFs.
- Paywalled-only PDFs.
- Copyright-unknown PDFs.
- Unauthorized reproductions.

## Required review before ingest

1. Source URL verification.
2. License verification.
3. Third-party material review.
4. Attribution requirement review.
5. Target venue policy review.
6. RAG metadata safety review.
