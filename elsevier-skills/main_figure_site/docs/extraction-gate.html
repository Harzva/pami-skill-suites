Extraction Gate

The extraction gate prevents unverified candidates from entering image extraction, public gallery hosting, or RAG indexing.

Unlock conditions

  1. Canonical DOI verified.
  2. Official publisher or repository source verified.
  3. Exact license text recorded.
  4. Permission and attribution requirements reviewed.
  5. Third-party material status checked.
  6. Manual reviewer decision recorded.

Unofficial research-assistant resource. Always verify live publisher, DOI, OA, license, permission, and third-party-material evidence before extraction or public reuse.
