Extraction Readiness Dashboard

Release: v2.3.0

The dashboard explains which seed figures can remain in the RAG index, which pending candidates are blocked, and why no pending slot can enter extraction before preflight passes.

Safety policy

This workflow is offline-safe by default. It does not download papers, bypass paywalls, use Sci-Hub, or treat pending candidates as approved OA sources.
