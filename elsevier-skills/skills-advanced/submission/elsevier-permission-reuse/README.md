# `elsevier-permission-reuse` skill

Component skill for **permissions for reused figures/tables/text, third-party material, CC license checks, and attribution wording** in Elsevier journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.

## Typical use

```text
@elsevier-permission-reuse Rewrite this section for Elsevier journal style and return a claim-evidence audit.
```
