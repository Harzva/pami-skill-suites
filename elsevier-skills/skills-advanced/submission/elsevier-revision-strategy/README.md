# Elsevier Revision Strategy

**Skill name:** `elsevier-revision-strategy`  
**Family:** `elsevier`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **reviewer response strategy, experiment triage, revised manuscript change plan, and cover letter update**.

## Typical commands

```text
Use elsevier-revision-strategy to audit this abstract for venue fit and reviewer risk.
Use elsevier-revision-strategy to rewrite my introduction in a more technical journal style.
Use elsevier-revision-strategy to produce a claim-evidence map and missing-experiment list.
Use elsevier-revision-strategy to draft a response letter from these reviewer comments.
```
