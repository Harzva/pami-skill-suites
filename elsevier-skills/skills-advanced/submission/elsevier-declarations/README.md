# Elsevier Declarations

**Skill name:** `elsevier-declarations`  
**Family:** `elsevier`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **declarations, ethics prompts, competing interests, funding, author contributions, and acknowledgements checklist**.

## Typical commands

```text
Use elsevier-declarations to audit this abstract for venue fit and reviewer risk.
Use elsevier-declarations to rewrite my introduction in a more technical journal style.
Use elsevier-declarations to produce a claim-evidence map and missing-experiment list.
Use elsevier-declarations to draft a response letter from these reviewer comments.
```
