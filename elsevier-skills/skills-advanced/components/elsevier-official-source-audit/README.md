# Elsevier Official Source Audit

Skill `elsevier-official-source-audit` for mapping workflow rules to Elsevier Guide for Authors, target journal instructions, LaTeX/source-package guidance, Highlights, Graphical Abstract, declarations, and GenAI policy checkpoints before making submission-readiness claims.

This is a product-level support skill: it routes to narrower skills, explains the source hierarchy, and turns official instructions plus open-access paper templates into maintainable writing workflows.

## Typical prompts

- Which skill should handle this manuscript?

- Compare my abstract against the bundled template-paper corpus structurally.
- Build a source-grounded submission-readiness audit.
