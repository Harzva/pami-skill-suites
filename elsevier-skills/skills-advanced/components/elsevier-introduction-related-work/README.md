# Elsevier Introduction + Related Work

**Skill name:** `elsevier-introduction-related-work`  
**Family:** `elsevier`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **application-aware motivation, literature positioning, and contribution framing**.

## Typical commands

```text
Use elsevier-introduction-related-work to audit this abstract for venue fit and reviewer risk.
Use elsevier-introduction-related-work to rewrite my introduction in a more technical journal style.
Use elsevier-introduction-related-work to produce a claim-evidence map and missing-experiment list.
Use elsevier-introduction-related-work to draft a response letter from these reviewer comments.
```
