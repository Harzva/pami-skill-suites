# Writing Patterns: Elsevier Title Author Metadata

## Good pattern

- Start from the reader's question.
- State the component's role in the manuscript.
- Use exact terminology and quantitative evidence.
- Preserve uncertainty and limitations.
- Cross-reference figures, tables, equations, algorithms, and supplementary files when relevant.

## Bad pattern

- Promotional language without evidence.
- Captions that merely repeat axis labels.
- Related-work claims that are not cited.
- Tables that duplicate text or hide important variables.
- Policy statements invented from memory.
