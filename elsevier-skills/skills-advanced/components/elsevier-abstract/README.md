# `elsevier-abstract` skill

Component skill for **abstract drafting, compression, claim discipline, and contribution evidence alignment** in Elsevier journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.

## Typical use

```text
@elsevier-abstract Rewrite this section for Elsevier journal style and return a claim-evidence audit.
```
