# Elsevier Graphical Abstract Planner

**Skill name:** `elsevier-graphical-abstract`  
**Family:** `elsevier`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **graphical abstract concept, layout, and captionless visual plan**.

## Typical commands

```text
Use elsevier-graphical-abstract to audit this abstract for venue fit and reviewer risk.
Use elsevier-graphical-abstract to rewrite my introduction in a more technical journal style.
Use elsevier-graphical-abstract to produce a claim-evidence map and missing-experiment list.
Use elsevier-graphical-abstract to draft a response letter from these reviewer comments.
```
