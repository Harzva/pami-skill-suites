# `elsevier-reference-integrity` skill

Component skill for **reference completeness, citation order, related-work coverage, and anti-fabrication checks** in Elsevier journal manuscripts.

Use this folder as a complete skill unit. Copy the entire folder, not just `SKILL.md`, because the references, checklists, and examples are part of the execution context.
