# Elsevier Statistical Significance

**Skill name:** `elsevier-statistical-significance`  
**Family:** `elsevier`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **statistical tests, confidence intervals, robustness, multi-run reporting, and effect-size interpretation**.

## Typical commands

```text
Use elsevier-statistical-significance to audit this abstract for venue fit and reviewer risk.
Use elsevier-statistical-significance to rewrite my introduction in a more technical journal style.
Use elsevier-statistical-significance to produce a claim-evidence map and missing-experiment list.
Use elsevier-statistical-significance to draft a response letter from these reviewer comments.
```
