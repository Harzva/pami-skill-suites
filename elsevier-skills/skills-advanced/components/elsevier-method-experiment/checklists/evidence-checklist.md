# Claim-Evidence Checklist

Use this table before rewriting or submission.

| Claim | Location | Evidence type | Status | Needed action |
|---|---|---|---|---|
|  | Abstract | result / citation / theory / assumption | supported / weak / unsupported |  |
|  | Introduction | gap / prior work / contribution | supported / weak / unsupported |  |
|  | Method | design rationale / proof / complexity | supported / weak / unsupported |  |
|  | Experiments | table / figure / statistical test | supported / weak / unsupported |  |
|  | Conclusion | implication / limitation | supported / weak / unsupported |  |

## Labeling rules

- Use **supported** only when evidence is present in the user's material.
- Use **needs citation** when the claim depends on external literature.
- Use **needs experiment** when the claim requires empirical verification.
- Use **soften** when evidence is incomplete but the idea is still plausible.
- Use **remove** when the claim is unsupported and unsafe to keep.
