# Elsevier LaTeX Template Audit

**Skill name:** `elsevier-latex-template-audit`  
**Family:** `elsevier`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **elsarticle-style source package, references, highlights, graphical abstract, and submission files**.

## Typical commands

```text
Use elsevier-latex-template-audit to audit this abstract for venue fit and reviewer risk.
Use elsevier-latex-template-audit to rewrite my introduction in a more technical journal style.
Use elsevier-latex-template-audit to produce a claim-evidence map and missing-experiment list.
Use elsevier-latex-template-audit to draft a response letter from these reviewer comments.
```
