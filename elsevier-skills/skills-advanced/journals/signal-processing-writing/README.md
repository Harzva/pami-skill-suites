# Signal Processing

**Skill name:** `signal-processing-writing`  
**Family:** `elsevier`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **signal processing theory, algorithms, audio/image/biomedical signals, filtering, estimation, and learning-based signal models**.

## Typical commands

```text
Use signal-processing-writing to audit this abstract for venue fit and reviewer risk.
Use signal-processing-writing to rewrite my introduction in a more technical journal style.
Use signal-processing-writing to produce a claim-evidence map and missing-experiment list.
Use signal-processing-writing to draft a response letter from these reviewer comments.
```
