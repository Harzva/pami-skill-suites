# Medical Image Analysis

**Skill name:** `medical-image-analysis-writing`  
**Family:** `elsevier`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **medical image computing, analysis, diagnosis, clinical imaging AI**.

## Typical commands

```text
Use medical-image-analysis-writing to audit this abstract for venue fit and reviewer risk.
Use medical-image-analysis-writing to rewrite my introduction in a more technical journal style.
Use medical-image-analysis-writing to produce a claim-evidence map and missing-experiment list.
Use medical-image-analysis-writing to draft a response letter from these reviewer comments.
```
