# Journal of Visual Communication and Image Representation

**Skill name:** `jvcir-writing`  
**Family:** `elsevier`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **visual communication, image/video representation, compression, retrieval, and media analysis**.

## Typical commands

```text
Use jvcir-writing to audit this abstract for venue fit and reviewer risk.
Use jvcir-writing to rewrite my introduction in a more technical journal style.
Use jvcir-writing to produce a claim-evidence map and missing-experiment list.
Use jvcir-writing to draft a response letter from these reviewer comments.
```
