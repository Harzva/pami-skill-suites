# Data & Knowledge Engineering

**Skill name:** `data-knowledge-engineering-writing`  
**Family:** `elsevier`  
**Version:** `0.2.0`  
**Status:** experimental, unofficial

## Purpose

This skill helps an AI assistant draft, revise, audit, and respond to academic manuscript materials for **data management, knowledge representation, databases, reasoning, information systems, and semantic methods**.

## Typical commands

```text
Use data-knowledge-engineering-writing to audit this abstract for venue fit and reviewer risk.
Use data-knowledge-engineering-writing to rewrite my introduction in a more technical journal style.
Use data-knowledge-engineering-writing to produce a claim-evidence map and missing-experiment list.
Use data-knowledge-engineering-writing to draft a response letter from these reviewer comments.
```
