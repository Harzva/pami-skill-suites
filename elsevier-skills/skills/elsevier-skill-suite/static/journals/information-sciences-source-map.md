1. Treat these links as verification entry points, not immutable rules.
2. For page limits, fees, license options, article types, and submission portals, verify live pages before advising the author.
3. If the live guide conflicts with this repository, the live official guide wins.
4. Do not invent author-agreement terms, copyrights, APCs, or license consequences from memory.
