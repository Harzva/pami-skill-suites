# Router risk map

| Risk | Router response |
|---|---|
| Wrong publisher family | Ask or infer conservatively; do not treat Elsevier PR as IEEE. |
| Overloaded task | Split into journal, component, and submission phases. |
| Policy-sensitive request | Route to official-source audit and author-agreement skills. |
| Many installed skills | Use registry-based selection, not a giant prompt. |
| Missing evidence | Route to claim-evidence, experiment, table, figure, and statistics skills. |
