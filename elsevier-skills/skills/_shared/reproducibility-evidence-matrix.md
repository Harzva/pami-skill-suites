# Reproducibility Evidence Matrix

| Area | Minimum useful detail |
|---|---|
| Data | source, version, splits, preprocessing, exclusions |
| Model/method | architecture, objective, hyperparameters, algorithm steps |
| Training | optimizer, schedule, epochs, batch size, seeds, hardware when relevant |
| Evaluation | metrics, confidence intervals/statistics when relevant, baselines, scripts |
| Artifacts | code/data/model availability, license, persistent link, access restrictions |
| Negative results | failure cases, sensitivity, limitations |
