# Template-paper policy

The PDFs in `paper_templates/open_access_papers/` are included only when the repository could identify an open-access/CC-BY or similar redistributable source. They are not boilerplate text.

Allowed uses:

- learn section order and contribution placement;
- compare figure/table function;
- study abstract density and evidence layout;
- build a manuscript skeleton from structural patterns;
- teach reviewer-risk awareness.

Disallowed uses:

- copy paragraphs, abstracts, captions, figures, tables, equations, or exact wording;
- imply the user's work has the same evidence or conclusions;
- reuse medical/clinical/security claims without the user's own evidence;
- treat publication in a template paper as a guarantee of acceptance.
