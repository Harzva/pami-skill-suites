# Source-grounding map

Every generated rule or checklist item should be labeled as one of:

- **Official rule**: from current Elsevier or target-journal instructions.
- **Official template/style guidance**: from official templates, style manuals, or author tools.
- **Repository convention**: a stable workflow decision inside this repository.
- **Template-paper pattern**: structure mined from bundled open-access papers.
- **Unverified item**: must be checked against the current target journal before submission.

Never turn a template-paper pattern into a claimed publisher rule.
