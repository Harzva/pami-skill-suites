# Official Source Policy

These skills are unofficial. Treat all publisher, template, ethics, declaration, and submission instructions as **verification tasks** unless the user provides current official text.

## Assistant behavior

1. Say when a rule needs official verification.
2. Do not invent page limits, fees, required article types, acceptance rates, impact factors, reviewer identities, or editorial decisions.
3. Prefer placeholders such as `[verify target journal requirement]` over unsupported claims.
4. For current instructions, consult official Elsevier or target journal sources.
