# Component Skill Index

The suite now includes granular skills for every high-risk part of a manuscript:

- Front matter: title, author metadata, abstract, keywords/index terms.
- Narrative: introduction, related work, contribution positioning, discussion, conclusion.
- Technical core: problem formulation, notation, method, equations, algorithms, theory/proofs.
- Evidence: datasets, baselines, metrics, experiments, ablations, statistics, robustness, limitations.
- Visual communication: tables, figures, captions, accessibility, graphical abstracts, supplementary visuals.
- Submission package: cover letter, metadata, declarations, data/code availability, permissions, copyright, open access, author agreement, final files, proofs.
- Revision: reviewer response matrix, revision diff, risk register, editorial screening.

Do not collapse these into one large writing prompt; separate skills make review, installation, testing, and official-source audits easier.
