# Elsevier Journal Taxonomy

## Core adapters in this repository

- `pr-writing`: Pattern Recognition.
- `prl-writing`: Pattern Recognition Letters.
- `cviu-writing`: Computer Vision and Image Understanding.
- `eswa-writing`: Expert Systems With Applications.
- `neurocomputing-writing`: Neurocomputing.
- `information-sciences-writing`: Information Sciences.
- `kbs-writing`: Knowledge-Based Systems.
- `applied-soft-computing-writing`: Applied Soft Computing.
- `artificial-intelligence-writing`: Artificial Intelligence.
- `medical-image-analysis-writing`: Medical Image Analysis.
- `image-and-vision-computing-writing`: Image and Vision Computing.

## Routing rule

If the task is broad Elsevier style, start with `elsevier-writing`. If a target journal is named, use the closest adapter after running the Elsevier submission-package checks.
