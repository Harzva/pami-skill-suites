## Repository policy

- Do not encode rigid word limits, graphical abstract requirements, declarations, fees, impact factors, acceptance rates, or editorial rules unless checked against official current sources.
- Elsevier requirements vary by journal; every skill should remind users to check the target journal Guide for Authors.
- Use this repository for drafting, auditing, and workflow support.
