Live Source Checks

Live checks are performed by scripts/check_official_links_live.py --live in a networked maintenance environment. Dry-run builds never claim unchecked URLs are healthy.

Offline-safe policy

Dry-run release builds must not mark unchecked sources as healthy. Mutable publication details require live verification before real submission advice.
