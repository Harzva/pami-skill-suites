# Manuscript Component Coverage

This repository now covers manuscript details at component level.

| Area | Covered by |
|---|---|
| Title / authors / metadata | `elsevier-title-author-metadata` |
| Abstract | `elsevier-abstract` |
| Keywords / Index Terms | `elsevier-keywords` |
| Introduction | `elsevier-introduction` |
| Related work | `elsevier-related-work` |
| Method / algorithm / theory | `elsevier-method-details`, `elsevier-algorithm-pseudocode`, `elsevier-theory-proof` |
| Experiments / baselines / metrics | `elsevier-experimental-design`, `elsevier-baseline-selection`, `elsevier-metrics-evaluation` |
| Ablations / statistics | `elsevier-ablation-study`, `elsevier-statistical-reporting` |
| Tables | `elsevier-table` |
| Figures | `elsevier-figure` |
| Captions | `elsevier-caption` |
| Discussion / limitations / conclusion | `elsevier-discussion`, `elsevier-limitations`, `elsevier-conclusion` |
| Supplementary / data / code | `elsevier-supplementary-files`, `elsevier-data-code-availability` |
| Author agreement / copyright / OA | `elsevier-author-agreement`, `elsevier-copyright-license`, `elsevier-open-access-decision` |
| Response / revision | `elsevier-reviewer-response-matrix`, `elsevier-revision-diff` |
