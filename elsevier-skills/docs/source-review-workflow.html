Source Review Workflow

How maintainers classify source trust and resolve human-review queue items.

Workflow

  1. Run python scripts/classify_source_trust.py.
  2. Run python scripts/build_human_review_queue.py.
  3. Review resources/human_review_queue.json.
  4. Patch official source records or staged journal fragments.
  5. Run python scripts/validate_source_trust.py and all release gates.

Rules

  • Publisher, society, and journal official sources outrank repository recommendations.
  • Template-paper observations are only structural mining aids.
  • Unverified candidates cannot be used for real submission advice.
  • Do not promote a candidate into default skills/.

Elsevier Skills is unofficial. Verify live publisher and journal pages before real submission advice.
