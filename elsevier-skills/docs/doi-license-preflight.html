DOI / License Preflight

Release: v2.3.0

Preflight records separate DOI lookup, OA source status, license candidate, and permission risk before any extraction task can proceed.

Safety policy

This workflow is offline-safe by default. It does not download papers, bypass paywalls, use Sci-Hub, or treat pending candidates as approved OA sources.

Required artifacts

data/doi_license_preflight_report.json
data/candidate_risk_scores.json

Maintainer command

python scripts/doi_license_preflight.py
