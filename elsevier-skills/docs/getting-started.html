Getting started

  1. Download the repository zip or a package under dist/.
  2. Install dist/default-skills.zip for normal use.
  3. Use dist/compact-suite.zip if you want only elsevier-skill-suite.
  4. Run python scripts/run_smoke_tests.py in developer mode.
Use elsevier-skill-suite to route my target-journal manuscript task. Select only necessary journal, component, and submission modules.

Elsevier Skills is unofficial and not affiliated with Elsevier. Verify current official instructions before real submission.
