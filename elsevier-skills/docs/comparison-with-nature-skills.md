# Comparison With Nature-style Skills

This repository complements Nature/CNS-oriented writing workflows.

| Dimension | Nature/CNS-style | Elsevier-style |
|---|---|---|
| Audience | broader scientific audience | domain experts and reviewers |
| Narrative | story-driven | claim-evidence-driven |
| Method detail | often concise in main text | usually more explicit |
| Experiments | impact and key validation | baselines, ablations, settings, robustness |
| Risk | overclaiming broad significance | insufficient technical evidence or package completeness |

Use this repository when the target output needs formal technical journal readiness rather than high-level editorial storytelling.
