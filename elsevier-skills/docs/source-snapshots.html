Source Snapshots

Source snapshots record official-source fields from the registry without fetching remote page content during offline release builds.

python scripts/snapshot_official_sources.py --label YYYYMMDD-live
python scripts/diff_source_snapshots.py --old resources/source_snapshots/old.json --new resources/source_snapshots/latest.json
