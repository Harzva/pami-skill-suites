one-skill distribution

Compact suite: one router skill that manages many modules.

The compact suite lives under dist/compact-suite/elsevier-skill-suite/. It has one public SKILL.md and many internal Markdown reference modules.

Install compact mode

cp -r dist/compact-suite/elsevier-skill-suite ~/.claude/skills/

Important design rule

Internal compact modules are not nested SKILL.md files. They are Markdown references under references/journals, references/components, and references/submission. This prevents host systems from confusing internal modules with independent skills.
