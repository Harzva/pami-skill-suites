# Compact Suite Distribution

The compact suite is generated by:

```bash
python scripts/build_compact_suite.py
```

Output:

```text
dist/compact-suite/elsevier-skill-suite/
├── SKILL.md
├── README.md
├── references/
│   ├── registry.json
│   ├── journals/
│   ├── components/
│   └── submission/
└── scripts/route_skill.py
```

Internal modules are Markdown references rather than nested `SKILL.md` files. This preserves a one-skill installation experience without breaking independent skill discovery in the full repository.
