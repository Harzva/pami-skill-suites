# Paper Mining Rubric

Applies to: Elsevier journal skill-suite evaluation tasks.

## Scoring scale

Each criterion is scored from 0 to 5.

- 0 = absent or unsafe
- 1 = present but mostly unusable
- 2 = partial and weak
- 3 = acceptable
- 4 = strong
- 5 = release-quality

## Criteria
1. **lawful source use.** The output must be specific, checkable, and aligned with the target journal task.
2. **structure-only extraction.** The output must be specific, checkable, and aligned with the target journal task.
3. **no copying of text/figures/tables.** The output must be specific, checkable, and aligned with the target journal task.
4. **reusable pattern quality.** The output must be specific, checkable, and aligned with the target journal task.
5. **copyright warning clarity.** The output must be specific, checkable, and aligned with the target journal task.

## Automatic gate

A sample output passes the static gate when it includes selected skills/modules, required warnings, a reviewer-risk checklist, and the structured output contract required by the task.

## Human review note

This rubric is a benchmark scaffold. It does not replace expert editorial review or live publisher/journal verification.
