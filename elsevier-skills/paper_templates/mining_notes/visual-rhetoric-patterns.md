# Visual Rhetoric Patterns

This note summarizes reusable visual-structure patterns learned from the local open-access visual asset corpus for Elsevier.

## Main figure routines

- Problem → method → evidence overview.
- Pipeline with semantically meaningful arrows.
- Taxonomy or design-space map for survey / positioning papers.
- Evidence-centered overview when the visual claim is experimental.

## Motivation figure routines

- Old paradigm vs. failure mode vs. proposed principle.
- Gap taxonomy with explicit boundary conditions.
- Toy example that explains why the proposed mechanism is necessary.

## Beautiful table routines

- Baseline comparison with clear protocol.
- Ablation staircase with component toggles and deltas.
- Dataset/protocol table for reproducibility.
- Efficiency/complexity trade-off table.

## Copyright safety

Use these as abstract patterns only. Do not copy visual layout, captions, numbers, labels, or graphics from template papers.
